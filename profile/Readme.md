## 1. 프로젝트 개요

### 프로젝트명  
CarsCommunity — 자동차 커뮤니티 & 튜닝 이미지 관리 플랫폼

### 프로젝트 기간  
2025년 11월 ~ 2025년 ing

### 프로젝트 목표
- 자동차 튜닝 정보를 구조적으로 저장·공유할 수 있는 커뮤니티 제공
  기존 커뮤니티의 단순 이미지 업로드 형태를 넘어서, 튜닝 이미지(2D) 뿐만 아니라 시뮬레이션 결과물(3D GLB)**까지 체계적으로 저장·관리할 수 있는 환경을 제공하는 것이 핵심 목표입니다.

- 사용자의 튜닝 데이터를 자산화할 수 있는 기능 구현
  사용자가 여러 SNS나 갤러리에 흩어져 있던 차량 사진·튜닝 스펙을 한 플랫폼에서 일관되게 관리할 수 있도록 데이터 구조를 설계했습니다. 이를 통해 사용자의 튜닝 히스토리를 "포트폴리오"처럼 축적할 수 있습니다.

- 상호작용(댓글·좋아요)을 통한 활발한 커뮤니티 기능 제공
  단순 게시판 기능이 아니라, 각 사용자 간의 피드백과 교류가 활발하게 이루어질 수 있도록 좋아요, 댓글, 알림 등 소셜 인터랙션 기능을 안정적으로 지원합니다. Redis를 활용한 빠른 인터랙션 처리를 도입하여 실시간성과 확장성도 확보했습니다.

### 주요 타겟 사용자
- 실제 튜닝을 즐기는 사용자(튜닝 러버)
  자신의 차량을 꾸민 2D/3D 이미지를 한 곳에 저장하고, 다른 유저와 공유하며 피드백을 받고 싶은 사용자에게 적합한 플랫폼입니다.

- 차량 커스터마이징 결과물을 저장하고 싶은 유저
  GLB/GLTF 같은 3D 모델을 저장할 수 있어, 기존 SNS나 커뮤니티로는 해결할 수 없던 실질적인 니즈를 충족시킵니다.

- 차량 정보·노하우·설치 팁을 공유하고 싶은 커뮤니티 유저
  차량 부품 선택 기준, 설치 경험, 튜닝 스펙 같은 정보를 구조적으로 공유할 수 있어,
  단순한 “자랑 게시판”이 아니라 지식 공유 기반 커뮤니티로 확장될 수 있습니다.

- 튜닝 스타일 탐색이나 차량별 셋업 참고가 필요한 사용자
  축적된 커뮤니티 데이터를 통해 사용자 맞춤 파츠 추천·튜닝 스타일 탐색이 가능해져 더 높은 활용성을 제공합니다.

## 2. 프로젝트 소개

### 프로젝트 배경 및 문제 정의

- 유저 니즈
  자동차 튜닝 유저들은 자신이 작업한 2D/3D 이미지, 차량 데이터, 파츠 정보를 한곳에 모아 체계적으로 저장하고 공유할 수 있는 환경을 필요로 한다.

- 정보 파편화
  튜닝 이미지, 파츠 데이터, 시뮬레이션 자료 등이 여러 플랫폼과 파일 형태에 흩어져 있어 효율적인 관리와 검색이 어렵다.

- 시뮬레이션 관리 부재
  GLB/GLTF 같은 3D 튜닝 시뮬레이션 결과물을 저장하거나 유저 간 공유할 수 있는 기능이 제공되지 않는다.

- 비표준화된 스펙 공유
  현재 튜닝 스펙 공유는 이미지 캡처나 수동 입력에 의존해 정확성, 재사용성, 확장성이 낮다.

- 통합 소통 환경 부족
  Q&A, 튜닝 자랑, 추천 등 다양한 유형의 소통이 여러 플랫폼에 분산되어 있어 하나의 커뮤니티 경험으로 제공되지 않는다.

### 기존 서비스의 한계 및 CarsCommunity의 차별화

- 단순 갤러리 기능
  대부분의 기존 커뮤니티는 사진 업로드 기능에만 집중하고 있어 3D 파일, 튜닝 스펙을 기록·관리하는 기능이 없다.

- 시뮬레이터와 분리된 구조
  튜닝 시뮬레이터에서 만든 결과물(GLB)을 커뮤니티 활동과 연계해 저장하거나 게시글에 자동 반영하는 기능이 없다.

- 파츠 정보 매핑 불가
  게시글 또는 이미지에서 사용된 파츠 정보를 자동으로 식별, 검색, 추천하는 기능이 없다.

- 데이터 재활용 부재
  유저가 공유한 튜닝 정보가 단순 게시글로만 소비되고, 검색·추천·분석 등으로 활용되지 못한다.

### CarsCommunity의 핵심 목표 및 개발 방향

- 3D 튜닝 결과물 저장·공유 기능 제공
  시뮬레이션한 튜닝 차량의 **3D 모델(GLB)**을 개인 저장소(슬롯)에 보관하고, 게시글 작성 시 첨부하여 공유할 수 있도록 한다.

- 통합 튜닝 스펙 관리
  사용자 설정 파츠 목록을 게시글과 자동 연결하고, 이를 기반으로 파츠 추천·쇼핑·장바구니·구매 연계 기능까지 확장 가능한 구조를 구축한다.

- 다양한 미디어 지원
  2D 이미지, 3D 모델 파일, JSON 기반의 스펙 데이터 등 다양한 형태의 미디어 파일 업로드를 지원하는 갤러리/게시판 환경을 제공한다.

- 검색·교류 활성화
  구조화된 튜닝 데이터 기반으로 유저들이 차량 모델별 튜닝 스타일, 파츠 종류, 유사 셋업을 쉽게 검색하고 소통할 수 있도록 한다.
